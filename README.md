<h1>💰 Projeto PooBank - Sistema Bancário em Java</h1>

<p>
O <strong>PooBank</strong> é um projeto desenvolvido em <strong>Java</strong> com foco na aplicação prática dos conceitos de 
<strong>Programação Orientada a Objetos (POO)</strong>. O sistema simula operações básicas de um banco, permitindo o gerenciamento de contas, clientes e transações financeiras.
</p>

<h2>🎯 Objetivo do Projeto</h2>
<p>
O principal objetivo deste projeto é consolidar conhecimentos em POO, incluindo:
</p>
<ul>
  <li>Encapsulamento</li>
  <li>Herança</li>
  <li>Polimorfismo</li>
  <li>Abstração</li>
</ul>

<h2>⚙️ Funcionalidades</h2>
<ul>
  <li>Criação de contas bancárias (Corrente e/ou Poupança)</li>
  <li>Cadastro de clientes</li>
  <li>Depósitos</li>
  <li>Saques com validação de saldo</li>
  <li>Transferências entre contas</li>
  <li>Consulta de saldo</li>
  <li>Histórico de transações</li>
</ul>

<h2>🏗️ Estrutura do Projeto</h2>
<p>
O sistema foi estruturado seguindo boas práticas de organização em camadas e separação de responsabilidades:
</p>
<ul>
  <li><strong>Model:</strong> Classes principais como Conta, Cliente e Transacao</li>
  <li><strong>Service:</strong> Regras de negócio (operações bancárias)</li>
  <li><strong>Main:</strong> Classe responsável pela execução e testes do sistema</li>
</ul>

<h2>🧠 Conceitos Aplicados</h2>
<ul>
  <li>Uso de classes e objetos para representar entidades do mundo real</li>
  <li>Encapsulamento com atributos privados e métodos públicos (getters/setters)</li>
  <li>Herança para especialização de tipos de conta</li>
  <li>Polimorfismo para comportamentos diferentes entre contas</li>
  <li>Validações de regras de negócio (ex: saldo insuficiente)</li>
</ul>

<h2>💻 Tecnologias Utilizadas</h2>
<ul>
  <li>Java (JDK 8+)</li>
  <li>Programação Orientada a Objetos</li>
</ul>

<h2>🚀 Como Executar</h2>
<pre>
1. Clone o repositório
2. Abra o projeto em sua IDE (IntelliJ, Eclipse ou VS Code)
3. Execute a classe principal (Main.java)
</pre>

<h2>📌 Considerações</h2>
<p>
Este projeto é ideal para fins acadêmicos e de aprendizado, servindo como base para evolução para sistemas mais complexos, 
como APIs REST, integração com banco de dados e interfaces gráficas.
</p>
